# 🛍️ E-Commerce Website (Frontend)

A modern and responsive **E-Commerce Website** built using **HTML, CSS, and JavaScript**. This project showcases a clean user interface for an online shopping platform with interactive features and a responsive design.

## 🚀 Features

* 🏠 Responsive Home Page
* 🛒 Product Listing Section
* 🔍 Product Search
* ❤️ Wishlist UI
* 🛍️ Shopping Cart Interface
* 📱 Fully Responsive Design
* 🎨 Modern and Clean UI
* ⚡ Smooth Animations & Hover Effects
* 📦 Product Cards with Images, Price, and Details

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
